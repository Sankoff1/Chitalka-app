# Внутренняя единица: регистрация корневого компонента Expo

**Родительский модуль:** `entry`  
**Файл кода:** `index.ts`

## Назначение

`registerRootComponent(App)` связывает компонент `App` из `./App` с точкой входа нативного приложения / Expo.

## Связи

- Зависит от успешного импорта `App.tsx` и порядка выше по файлу: [`entry-04-react-native-screens.md`](./entry-04-react-native-screens.md).
- Expo документация: корректная настройка окружения для dev client и Expo Go.

## Риски для агентов

`main` в `package.json` должен указывать на `index.ts`. Не регистрировать второй корень.
