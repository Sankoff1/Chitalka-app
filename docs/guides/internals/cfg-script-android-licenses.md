# Внутренняя единица: скрипт Android SDK

**Родительский модуль:** `scripts-android-licenses`  
**Файл:** `scripts/accept-android-licenses-and-sdk.ps1`

## Назначение

Вспомогательный PowerShell-скрипт для локальной машины разработчика: приём лицензий Android SDK / подготовка окружения (см. комментарии внутри файла).

## Связи

Не вызывается из runtime приложения; только из dev/CI вручную.

## Риски для агентов

Пути к SDK зависят от установки Android Studio — не хардкодить в приложении.
