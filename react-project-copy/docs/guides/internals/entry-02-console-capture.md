# Внутренняя единица: подключение перехвата консоли

**Родительский модуль:** `entry` — `index.ts`  
**Файл кода:** `src/debug/installConsoleCapture.ts` (импортируется вторым в `index.ts`)

## Назначение

При загрузке модуля вызывается `installConsoleCapture()`: обёртки вокруг `console.log|info|warn|error|debug` дублируют сообщения в буфер [`debug-log-buffer.md`](./debug-log-buffer.md).

## Инварианты

Идемпотентность через флаг на `globalThis` — повторный импорт не дублирует обёртки.

## Связи

- Используется экраном [`screen-debug-logs.md`](./screen-debug-logs.md).
- Не зависит от React-дерева.

## Риски для агентов

Отключение импорта уберёт логи из буфера (в т.ч. в release), но не сломает приложение.
