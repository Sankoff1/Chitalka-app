# Внутренняя единица: первый импорт gesture-handler

**Родительский модуль:** `entry` — `index.ts`  
**Файл кода:** `index.ts` (первая строка)

## Назначение

`import 'react-native-gesture-handler'` должен выполниться **до** любого рендера, использующего жесты навигации (drawer, stack). Это требование библиотеки.

## Поведение

Побочный эффект на уровне модуля: регистрация нативных обработчиков жестов.

## Связи

- Предшествует импорту `installConsoleCapture` и `App`.
- Не импортирует код проекта.

## Риски для агентов

Не переставлять ниже других импортов из RN-навигации без проверки drawer/stack. Не удалять при использовании `@react-navigation/drawer`.
