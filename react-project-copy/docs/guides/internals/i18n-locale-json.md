# Внутренняя единица: файлы локалей JSON

**Родительский модуль:** `i18n-locale-*`  
**Файлы:** `src/i18n/locales/ru.json`, `src/i18n/locales/en.json`

## Правила для агентов

- Любой ключ, используемый в `t('path')`, должен существовать **в обоих** файлах (или сознательно fallback на ru в коде каталога).
- Вложенность объектов отражает dot-path.
